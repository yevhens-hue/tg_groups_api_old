# 💻 Инструкция по установке на новом компьютере

## Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/yevhens-hue/mirror_finder.git
cd mirror_finder
```

### 2. Автоматическая установка

```bash
./setup_new_computer.sh
```

Скрипт автоматически установит все зависимости и настроит окружение.

---

## Ручная установка

Если автоматический скрипт не работает, выполните шаги вручную:

### Шаг 1: Проверка Python

```bash
python3 --version
# Должно быть Python 3.8 или выше
```

Если Python не установлен:
- **macOS**: `brew install python3`
- **Linux**: `sudo apt install python3 python3-pip`
- **Windows**: Скачайте с https://www.python.org/

### Шаг 2: Установка зависимостей

#### Indian Gambling Parser

```bash
cd indian_gambling_parser
pip3 install -r requirements.txt
python3 -m playwright install chromium
cd ..
```

#### Mirror Finder

```bash
cd mirror_finder
pip3 install -r requirements.txt
cd ..
```

#### Mirrors API

```bash
cd mirrors_api
pip3 install -r requirements.txt
cd ..
```

### Шаг 3: Создание директорий

```bash
# Для indian_gambling_parser
mkdir -p indian_gambling_parser/screenshots
mkdir -p indian_gambling_parser/storage_states
mkdir -p indian_gambling_parser/traces
```

### Шаг 4: Настройка Google Sheets (опционально)

Если нужен экспорт в Google Sheets:

1. **Создайте Service Account:**
   - Перейдите: https://console.cloud.google.com/
   - Создайте проект или выберите существующий
   - APIs & Services → Library → Включите Google Sheets API и Google Drive API
   - APIs & Services → Credentials → Create Credentials → Service Account
   - Создайте аккаунт и скачайте JSON ключ

2. **Сохраните credentials:**
   ```bash
   # Скопируйте скачанный JSON файл
   cp ~/Downloads/your-service-account.json indian_gambling_parser/google_credentials.json
   ```

3. **Предоставьте доступ к Google Таблице:**
   - Откройте JSON файл, найдите `client_email`
   - Откройте Google Таблицу
   - Нажмите "Поделиться" и добавьте email из JSON с правами "Редактор"

Подробная инструкция: `indian_gambling_parser/GOOGLE_SHEETS_SETUP.md`

## Запуск проектов

### Indian Gambling Parser

```bash
cd indian_gambling_parser
python3 main_parser_playwright.py --merchant 1xbet --url https://indian.1xbet.com
```

**Параметры:**
- `--merchant` - имя мерчанта (1xbet, и т.д.)
- `--url` - URL сайта
- `--headless` - запуск без окна браузера (опционально)

**Документация:** `indian_gambling_parser/ЗАПУСК.md`

### Mirror Finder

```bash
cd mirror_finder
python3 main_parser.py --help
```

### Mirrors API

```bash
cd mirrors_api
python3 api.py
# API будет доступен на http://localhost:8000
```

## Решение проблем

### Ошибка: "Python not found"

**Решение:**
- Установите Python 3.8+ с официального сайта
- Убедитесь, что `python3` доступен в PATH

### Ошибка: "pip not found"

**Решение:**
```bash
python3 -m ensurepip --upgrade
```

### Ошибка: "Playwright browser not found"

**Решение:**
```bash
python3 -m playwright install chromium
```

### Ошибка: "Permission denied" при установке пакетов

**Решение:**
```bash
# Используйте --user флаг
pip3 install --user -r requirements.txt
```

### Ошибка: "ModuleNotFoundError"

**Решение:**
```bash
# Убедитесь, что все зависимости установлены
pip3 install -r requirements.txt
```

### Ошибка: "Google Sheets API error"

**Решение:**
- Проверьте наличие `google_credentials.json`
- Убедитесь, что Service Account имеет доступ к таблице
- Проверьте, что включены Google Sheets API и Google Drive API
