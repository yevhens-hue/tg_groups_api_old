# Анализ перспективных стран для процессинга гемблинговых мерчантов

> **ВАЖНО**: Для данного анализа приоритет отдается странам с **отсутствием лицензирования** и **серой зоной**, где работают **офшорные мерчанты**. Легализованные рынки считаются менее приоритетными.

## Краткое резюме

### ТОП-2 приоритетные страны (серые зоны):

1. **ИНДОНЕЗИЯ** 🇮🇩 - $8,5 млрд прогноз, 275+ млн населения, полный запрет = множество офшорных мерчантов
2. **ТАИЛАНД** 🇹🇭 - $4,75 млрд GGR, 70+ млн населения, множество офшорных мерчантов в серой зоне

### Дополнительные перспективные страны:
- **ВЬЕТНАМ** - растущий рынок, множество офшорных мерчантов
- **ПЕРУ** - растущий рынок, пока регуляция не установлена
- **НИГЕРИЯ** - огромный рынок (200+ млн), требует изучения

## Методология оценки

### Критерии оценки:
1. **Отсутствие лицензирования / Серая зона** - ⭐ ПРЕИМУЩЕСТВО: отсутствие четкого регулирования создает возможности для офшорных мерчантов
2. **Количество офшорных мерчантов** - количество операторов, работающих в серой зоне (чем больше, тем лучше)
3. **Суммарный трафик** - объем интернет-трафика и вовлеченность пользователей
4. **Размер рынка (GGR)** - валовой доход от гемблинга
5. **Население и интернет-проникновение** - потенциальная аудитория
6. **Экономическая стабильность** - стабильность экономики и платежеспособность
7. **Уровень блокировок** - насколько активно власти блокируют сайты (чем меньше блокировок, тем лучше)

### Исключенные страны:
- **TIER 1**: США, Великобритания, Канада, Австралия, Германия, Франция, Италия, Испания, Нидерланды, Швеция, Норвегия, Дания, Швейцария, Австрия
- **Уже присутствуем**: Индия, Аргентина, Египет, Колумбия, Пакистан
- **Исключены из анализа**: Бангладеш
