# Работа с сохраненной сессией

## Первый запуск (логин и сохранение сессии)

Запустите скрипт `first_login.py` один раз, чтобы залогиниться и сохранить сессию:

```bash
python3 first_login.py
```

Этот скрипт:
- Откроет браузер и выполнит логин
- Сохранит cookies и session в `.playwright_context_ar/`
- После этого можно использовать автоматический режим

## Последующие запуски (из кэша)

После первого логина используйте `run_parser_ar.py` - он автоматически определит наличие сохраненной сессии:

```bash
python3 run_parser_ar.py
```

Скрипт автоматически:
- Обнаружит сохраненную сессию в `.playwright_context_ar/`
- Использует `skip_login=True` для работы без логина
- Будет работать быстрее, так как не нужно каждый раз логиниться

## Через API

При вызове API также используйте `skip_login=True` после первого логина:

```python
data = {
    "email": "perymury78@gmail.com",
    "password": '%m^%G5"}4m',
    "base_url": "https://1win.lat/",
    "use_persistent_context": True,
    "skip_login": True,  # Используем сохраненную сессию
    "wait_seconds": 30
}
```

## Перелогин

Если нужно перелогиниться, просто удалите папку с сессией:

```bash
rm -rf .playwright_context_ar/
```

После этого при следующем запуске будет выполнен новый логин.
