# Исправление: Упрощенное ожидание формы (как работало раньше)

## Проблема

После клика на кнопку Deposit форма с данными (CVU, Recipient) не появлялась, потому что код использовал сложные проверки, которые не срабатывали.

## Решение

Применен упрощенный подход, который работал раньше (когда все было успешно):
- После клика на Deposit код просто ждет 30 секунд
- Затем переходит к извлечению данных
- Это соответствует подходу, который работал вчера (2026-01-10 21:33:35)

## Изменения

### 1. Исправлен баг с `fiat_selected`

**Файл:** `services/payment_parser_ar.py`, строка ~2430

**Проблема:** Когда `deposit_button_clicked_early = True`, код устанавливал `fiat_selected = True`, но затем сразу перезаписывал его на `False`.

**Исправление:**
```python
# Было:
if deposit_button_clicked_early:
    fiat_selected = True

fiat_selected = False  # ❌ Перезаписывалось!

# Стало:
if deposit_button_clicked_early:
    fiat_selected = True
else:
    fiat_selected = False  # ✅ Правильно
```

### 2. Упрощено ожидание формы

**Файл:** `services/payment_parser_ar.py`, строки ~3530-3536

**Изменение:** После клика на Deposit код теперь просто ждет 30 секунд (как работало раньше), вместо сложных проверок формы.

```python
# Упрощенное ожидание формы: просто ждем фиксированное время (как работало раньше)
# После клика на Deposit форма должна появиться через 20-30 секунд
if deposit_button_clicked or deposit_button_clicked_early:
    logger.info("payment_parser_simple_form_wait_after_deposit", wait_seconds=30)
    await page.wait_for_timeout(30000)  # Ждем 30 секунд для загрузки формы (как работало вчера)
    form_appeared = True  # Предполагаем, что форма появилась после ожидания
    logger.info("payment_parser_form_wait_completed_proceeding_to_extraction")
    elapsed_total = 30
```

## Как это работает

1. ✅ Кнопка Deposit кликается (раньше или позже)
2. ✅ Код ждет 30 секунд для загрузки формы
3. ✅ Устанавливается `form_appeared = True`
4. ✅ Переход к извлечению данных (CVU, Recipient)

## Ожидаемый результат

После применения этого исправления:
- ✅ Код будет ждать 30 секунд после клика на Deposit
- ✅ Форма должна появиться за это время
- ✅ CVU и Recipient должны быть найдены (если форма появилась)
