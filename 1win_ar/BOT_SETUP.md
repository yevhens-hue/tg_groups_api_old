## 🛠️ Устранение неполадок

### Бот не отвечает

1. Проверьте, что токен установлен:
   ```bash
   echo $TELEGRAM_BOT_TOKEN
   ```

2. Проверьте, что бот запущен:
   ```bash
   python bot.py
   ```

3. Проверьте логи на наличие ошибок

### Ошибка экспорта в Google Sheets

1. Убедитесь, что файл `token.json` существует
2. Проверьте права доступа к Google Sheets API
3. Убедитесь, что токен не истек
