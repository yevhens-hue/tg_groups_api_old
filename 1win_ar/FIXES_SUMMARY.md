**Стало:** 5 секунд после клика + активная проверка в течение 10 секунд

**Код:**
- Увеличил `await page.wait_for_timeout(2000)` до `await page.wait_for_timeout(5000)`
- Добавил активную проверку каждую секунду в течение 10 секунд после клика

### 3. Улучшенное ожидание формы

**Было:** 45 секунд, проверка каждую секунду
**Стало:** 60 секунд, проверка каждые 2 секунды + активная проверка после клика

**Изменения:**
- Увеличено общее время ожидания до 60 секунд
- Интервал проверки изменен с 1 до 2 секунд (реже, но точнее)
- Добавлена активная проверка формы каждую секунду в течение 10 секунд после клика

### 4. Улучшенная проверка формы

**Было:** Проверка только полных полей (hasAllFields)
**Стало:** Проверка даже частичного наличия данных

**Новая логика:**
```javascript
// Учитываем даже частичное наличие данных
has_any_data = hasCvuLabel || hasRecipientLabel || hasLongNumber || hasWaitingText

// Если есть данные и текст длинный (> 1200 символов) - форма появилась
if (hasAllFields || (hasCvuLabel && hasRecipientLabel) || 
    (hasCopyBankDetails && hasLongNumber) || 
    (has_any_data && textLength > 1200)) {
    // Форма найдена!
}
```
