# 📁 Структура проекта "Парсер провайдеров гемблинговых сайтов Индии"

## Основные файлы

### Код парсера
- `provider_parser_playwright.py` - Основной парсер на Playwright
  - Логин на сайты мерчантов
  - Переход на кэшир
  - Поиск провайдеров в E-WALLETS
  - Обработка iframe, popup, network requests
  - Human-in-the-loop для капчи

- `storage.py` - Работа с данными
  - SQLite база данных
  - Экспорт в XLSX
  - Нормализация доменов

- `main_parser_playwright.py` - CLI интерфейс
  - Запуск парсера
  - Просмотр результатов
  - Экспорт данных

- `merchants_config.py` - Конфигурация
  - Креденшиалы для входа
  - CSS селекторы для каждого мерчанта
  - Настройки account_type

### Документация

- `README.md` - Главный README проекта
- `README_PLAYWRIGHT.md` - Полная документация по Playwright парсеру
- `HUMAN_IN_THE_LOOP.md` - Документация по механизму human-in-the-loop
- `QUICKSTART.md` - Быстрый старт
- `ИНСТРУКЦИЯ.md` - Инструкция на русском языке
- `ЧТО_ДЕЛАТЬ.txt` - Краткая инструкция

### Данные и результаты

- `providers_data.db` - SQLite база данных с результатами
- `providers_data.xlsx` - Excel экспорт (создаётся автоматически)
- `screenshots/` - Скриншоты платёжных форм провайдеров
- `storage_states/` - Сохранённые cookies (storageState) для каждого мерчанта
- `traces/` - Trace файлы Playwright для отладки

### Конфигурация

- `requirements.txt` - Python зависимости
- `.gitignore` - Игнорируемые файлы

## Как использовать

1. Перейдите в папку: `cd indian_gambling_parser`
2. Установите зависимости: `pip install -r requirements.txt && playwright install chromium`
3. Настройте креденшиалы в `merchants_config.py`
4. Запустите: `python3 main_parser_playwright.py --merchant 1xbet --url https://indian.1xbet.com`

## Результаты работы

После запуска парсера вы получите:
- ✅ Таблицу со всеми провайдерами (БД + Excel)
- ✅ Скриншоты платёжных форм
- ✅ Сохранённые cookies для следующих запусков
