### URL Параметры

Фильтры сохраняются в URL как query параметры:
- `?merchant=1win`
- `?accountType=FTD`
- `?page=2&limit=50`
- `?sortBy=timestamp_utc&sortOrder=desc`
- `?search=paytm`

### Синхронизация

1. **При изменении фильтров** → обновляется URL
2. **При изменении URL** (навигация назад/вперед) → обновляются фильтры
3. **При загрузке страницы** → фильтры читаются из URL
4. **При изменении пагинации** → обновляется URL
5. **При изменении сортировки** → обновляется URL

## 🚀 Использование

### Поделиться ссылкой с фильтрами

Теперь пользователи могут:
1. Применить фильтры на странице
2. Скопировать URL из адресной строки
3. Отправить ссылку другому пользователю
4. При открытии ссылки фильтры автоматически применятся

**Пример:**
```
https://example.com/?merchant=1win&accountType=FTD&page=2&limit=50
```

### Навигация

- Кнопки "Назад"/"Вперед" браузера работают с фильтрами
- История браузера сохраняет состояние фильтров
- Можно добавить в закладки страницу с фильтрами
