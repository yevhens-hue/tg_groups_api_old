## 🚀 Как увидеть новые функции

### 1. Перезапустите сервисы

**Backend:**
```bash
cd web_service/backend
python3 start.py
```

**Frontend:**
```bash
cd web_service/frontend
npm run dev
```

### 2. Обновите страницу в браузере

Hard refresh: **Cmd+Shift+R** (Mac) или **Ctrl+Shift+R** (Windows)

- ✅ **Индикатор WebSocket** - 🟢 Real-time / 🔴 Offline (правый верхний угол)
