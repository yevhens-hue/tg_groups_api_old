## 🎨 Цветовая палитра

### Основные цвета:
- **Primary (Синий):** `#90caf9` - основные элементы, ссылки
- **Secondary (Фиолетовый):** `#ce93d8` - вторичные элементы
- **Success (Зеленый):** `#66bb6a` - успешные операции
- **Warning (Оранжевый):** `#ffa726` - предупреждения
- **Error (Красный):** `#ef5350` - ошибки
- **Info (Голубой):** `#42a5f5` - информационные сообщения

### Фоны:
- **Background:** `#121212` - основной фон
- **Paper:** `#1e1e1e` - фон карточек и панелей
- **Cards:** Градиенты с прозрачностью `rgba(..., 0.15)`

---

## 🌟 Визуальные эффекты

### Градиенты:
- Заголовки: `linear-gradient(135deg, #90caf9 0%, #ce93d8 100%)`
- Кнопки: `linear-gradient(135deg, #90caf9 0%, #42a5f5 100%)`
- Карточки: Градиентные фоны с прозрачностью

### Тени:
- Карточки: `0 4px 20px rgba(0, 0, 0, 0.3)`
- Hover: `0 6px 25px rgba(0, 0, 0, 0.4)`
- Кнопки: `0 2px 8px rgba(0, 0, 0, 0.3)`

### Переходы:
- Все интерактивные элементы: `0.2s ease-in-out`
- Hover эффекты: Плавные трансформации

### Скругления:
- Карточки и Paper: `12px`
- Кнопки и TextField: `8px`
- Таблица: `8px`
