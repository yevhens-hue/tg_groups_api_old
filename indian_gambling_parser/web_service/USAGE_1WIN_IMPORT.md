## 🎯 Способ 1: Через веб-интерфейс (рекомендуется)


### Шаг 2: Найдите блок "Импорт данных из Google Sheets"
Блок расположен **выше фильтров**, в верхней части страницы.

### Шаг 3: Настройте параметры
- **GID листа**: `396039446` (уже заполнено по умолчанию для 1win IN)

1. Нажмите кнопку **"Предпросмотр"**
2. Откроется модальное окно с первыми 10 записями
3. Проверьте, что данные правильно распарсены

1. Нажмите кнопку **"Импортировать"**
2. Дождитесь завершения (появится сообщение с результатами)
3. Страница автоматически обновится через 2 секунды
4. Новые данные появятся в таблице!

## 🐛 Troubleshooting

### Ошибка: "Файл credentials не найден"
**Решение:**
1. Создайте Service Account в Google Cloud Console
2. Скачайте JSON ключ
3. Сохраните как `google_credentials.json` в корне проекта
4. См. `GOOGLE_SHEETS_SETUP.md`

### Ошибка: "Лист не найден"
**Решение:**
1. Проверьте GID в URL таблицы: `?gid=396039446`
2. Убедитесь, что Service Account имеет доступ к таблице
3. Попробуйте использовать название листа: `sheet_name=Scraper (PY)`

### Нет данных после импорта
**Решение:**
1. Проверьте предпросмотр: должен показать данные
2. Проверьте логи backend на ошибки
3. Убедитесь, что в таблице есть данные для 1win

### Дубликаты не импортируются
**Это нормально!** Дубликаты автоматически пропускаются (проверка по merchant_domain + provider_domain + account_type).
