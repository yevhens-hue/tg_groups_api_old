## ⚡ Быстрый старт

```bash
# Backend
cd web_service/backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000

# Frontend
cd web_service/frontend
npm install
npm run dev
```

Откройте: http://localhost:5173
