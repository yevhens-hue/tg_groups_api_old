## 🔄 Поток данных - Детальная схема

### 1. Сбор данных (Парсер → БД)

```
┌──────────────┐
│   Playwright │
│   Browser    │
└──────┬───────┘
       │
       │ 1. Логин
       ▼
┌──────────────┐
│   Мерчант    │
│    сайт      │
└──────┬───────┘
       │
       │ 2. Cashier
       ▼
┌──────────────┐
│   E-WALLETS  │
│    секция    │
└──────┬───────┘
       │
       │ 3. Для каждого провайдера
       ▼
┌──────────────────────────┐
│  Обработка провайдера:   │
│  - Клик                  │
│  - Извлечение URL        │
│  - Скриншот              │
│  - Детекция особенностей │
└──────┬───────────────────┘
       │
       │ 4. Сохранение
       ▼
┌──────────────┐      ┌──────────────┐
│  SQLite БД   │◄────►│  XLSX файл   │
│ providers_   │      │ providers_   │
│ data.db      │      │ data.xlsx    │
└──────────────┘      └──────────────┘
```

### 2. Просмотр данных (Frontend → Backend → БД)

```
┌──────────────┐
│   Frontend   │
│   (React)    │
└──────┬───────┘
       │
       │ GET /api/providers?merchant=1win&skip=0&limit=50
       ▼
┌──────────────┐
│   Backend    │
│  (FastAPI)   │
└──────┬───────┘
       │
       │ StorageAdapter.get_all_providers()
       ▼
┌──────────────┐
│   SQLite     │
│    БД        │
└──────┬───────┘
       │
       │ SELECT * FROM providers WHERE merchant='1win' LIMIT 50
       │
       ▼
┌──────────────┐
│   JSON Data  │
│  {items: [], │
│   total: 100}│
└──────┬───────┘
       │
       │ HTTP Response
       ▼
┌──────────────┐
│   Frontend   │
│   отображает │
│   в DataTable│
└──────────────┘
```

### 3. Real-time обновления (WebSocket)

```
┌──────────────┐                    ┌──────────────┐
│   Backend    │                    │   Frontend   │
│              │                    │              │
│ Connection   │◄─── WebSocket ────►│ useWebSocket │
│ Manager      │     подключение    │    hook      │
└──────┬───────┘                    └──────────────┘
       │
       │ Каждые 5 секунд
       ▼
┌──────────────────────────┐
│ Проверка изменений:      │
│ 1. Читает данные из БД   │
│ 2. Вычисляет hash        │
│ 3. Сравнивает с last_hash│
└──────┬───────────────────┘
       │
       │ Если изменилось
       ▼
┌──────────────────────────┐
│ Отправка обновления:     │
│ broadcast({              │
│   type: "data_updated",  │
│   statistics: {...},     │
│   total_providers: 100   │
│ })                       │
└──────┬───────────────────┘
       │
       │ WebSocket message
       ▼
┌──────────────┐
│   Frontend   │
│              │
│ - Получает   │
│   сообщение  │
│ - Invalidate │
│   queries    │
│ - Обновляет  │
│   UI         │
└──────────────┘
```
