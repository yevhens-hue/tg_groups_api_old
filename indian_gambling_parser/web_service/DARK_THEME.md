## 🎨 Визуальные улучшения

### Градиентный текст
Заголовки используют градиент (синий → фиолетовый):
```css
background: linear-gradient(135deg, #90caf9 0%, #ce93d8 100%);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```

### Карточки метрик
Каждая карточка имеет уникальный градиент:
- Всего провайдеров: Синий градиент
- Новых сегодня: Зеленый градиент
- За 7 дней: Голубой градиент
- Активных merchants: Оранжевый градиент

### Скругления
- Карточки: 12px
- Кнопки: 8px
- TextField: 8px
- Таблица: 8px

### Тени и эффекты
- Мягкие тени для глубины
- Hover эффекты с подъемом карточек
- Плавные переходы (0.2s)

## 🔧 Кастомизация

### Изменение цветовой схемы

В `App.tsx` можно настроить тему:

```typescript
const theme = createTheme({
  palette: {
    mode: 'dark',
    primary: {
      main: '#90caf9', // Измените на свой цвет
    },
    // ...
  },
});
```

### Стилизация компонентов

Все компоненты используют sx prop для стилизации:
- Можно легко изменить цвета
- Настроить градиенты
- Изменить размеры и отступы
