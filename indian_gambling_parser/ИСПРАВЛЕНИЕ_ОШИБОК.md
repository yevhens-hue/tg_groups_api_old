# 🔧 Исправление ошибок

## Ошибка: Executable doesn't exist

```
playwright._impl._errors.Error: BrowserType.launch: Executable doesn't exist
```

### Решение:

```bash
# В виртуальном окружении выполните:
playwright install chromium
```

Или установите все браузеры:

```bash
playwright install
```

## Ошибка: ModuleNotFoundError: No module named 'playwright'

### Решение:

```bash
# В виртуальном окружении:
pip install playwright
playwright install chromium
```

## Полная установка с нуля:

```bash
# 1. Активируйте виртуальное окружение
source .venv/bin/activate

# 2. Установите все зависимости
pip install -r requirements.txt

# 3. Установите браузеры Playwright
playwright install chromium

# 4. Проверьте установку
python3 main_parser_playwright.py --list-merchants
```

## Если проблема остаётся:

1. Убедитесь, что используете правильный Python:
   ```bash
   which python3
   ```

2. Проверьте, что виртуальное окружение активировано:
   ```bash
   echo $VIRTUAL_ENV
   ```

3. Переустановите Playwright:
   ```bash
   pip uninstall playwright -y
   pip install playwright
   playwright install chromium
   ```
