## 📊 Структура данных

Сервис работает с данными из таблицы `providers`:

| Поле | Тип | Описание |
|------|-----|----------|
| merchant | TEXT | ID мерчанта |
| merchant_domain | TEXT | Домен мерчанта |
| account_type | TEXT | Тип аккаунта (player/admin/agent) |
| provider_domain | TEXT | Домен провайдера (eTLD+1) |
| provider_name | TEXT | Имя провайдера |
| provider_entry_url | TEXT | Первый внешний URL |
| final_url | TEXT | Финальный URL после редиректов |
| cashier_url | TEXT | URL страницы кэшира |
| screenshot_path | TEXT | Путь к скриншоту |
| detected_in | TEXT | Где обнаружен |
| payment_method | TEXT | Метод оплаты |
| is_iframe | INTEGER | Форма в iframe (0/1) |
| requires_otp | INTEGER | Требует OTP (0/1) |
| blocked_by_geo | INTEGER | Заблокирован по гео (0/1) |
| captcha_present | INTEGER | Присутствует капча (0/1) |
| timestamp_utc | TIMESTAMP | Время создания |

## 🐛 Troubleshooting

### Ошибка импорта storage
```bash
# Убедитесь, что вы в корневой директории проекта
cd /Users/yevhen.shaforostov/indian_gambling_parser
streamlit run web_service_streamlit.py
```

### Нет данных в таблице
- Убедитесь, что парсер уже был запущен
- Проверьте наличие файла `providers_data.db`
- Запустите парсер: `python main_parser_playwright.py --show-results`

### Проблемы с скриншотами
- Убедитесь, что путь к скриншотам корректен
- Проверьте права доступа к файлам
- Используйте абсолютные пути при необходимости
