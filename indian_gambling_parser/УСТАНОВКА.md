# 🔧 Установка зависимостей

## Проблема: ModuleNotFoundError: No module named 'playwright'

Если вы видите эту ошибку, значит зависимости не установлены в вашем виртуальном окружении.

## Решение:

### Вариант 1: Установка в виртуальное окружение (рекомендуется)

```bash
# Убедитесь, что виртуальное окружение активировано
source .venv/bin/activate  # или активируйте ваше окружение

# Установите зависимости
pip install -r requirements.txt

# Установите браузеры Playwright
playwright install chromium
```

### Вариант 2: Установка в системный Python

```bash
# Установите зависимости
pip3 install -r requirements.txt

# Установите браузеры Playwright
playwright install chromium
```

### Вариант 3: Если используете виртуальное окружение

```bash
# Активируйте виртуальное окружение
source .venv/bin/activate

# Установите зависимости
pip install playwright beautifulsoup4 requests fastapi uvicorn pydantic openpyxl pandas tldextract

# Установите браузеры
playwright install chromium
```

## Проверка установки:

```bash
# Проверьте, что Playwright установлен
python3 -c "import playwright; print('✓ Playwright установлен')"

# Проверьте список мерчантов
python3 main_parser_playwright.py --list-merchants
```

## Если проблема остаётся:

1. Убедитесь, что используете правильный Python:
   ```bash
   which python3
   which pip3
   ```

2. Проверьте, что виртуальное окружение активировано:
   ```bash
   echo $VIRTUAL_ENV
   ```

3. Переустановите зависимости:
   ```bash
   pip uninstall playwright -y
   pip install playwright
   playwright install chromium
   ```
